Em **APIs** você conecta serviços externos (sistemas internos, ferramentas de terceiros) para que os agentes possam usá-los durante o trabalho — buscar um dado, criar um registro, disparar uma ação.

<Frame caption="📸 Screenshot a inserir: tela de APIs (Recursos → APIs).">
  <img src="/images/placeholder.svg" alt="APIs" />
</Frame>

## O que dá para fazer

- Registrar uma conexão com um serviço (endereço + credenciais).
- Tornar essa conexão disponível como **ferramenta** para os agentes escolhidos.
- Manter as credenciais **guardadas com segurança** — os agentes usam sem expor a chave.

## Como conectar uma API

<Steps>
  <Step title="Abra Recursos → APIs e clique em adicionar conexão" />
  <Step title="Informe o serviço e as credenciais">
    As credenciais ficam protegidas. Você não precisa colá-las em conversas.
  </Step>
  <Step title="Libere para os agentes certos">
    Só os agentes que precisam recebem acesso à conexão.
  </Step>
</Steps>

<Warning>
Nunca cole chaves de API ou senhas dentro de uma conversa com o agente. Cadastre-as aqui, no lugar seguro. [Ver Segurança & Acessos →](/pt/administracao/seguranca-e-acessos)
</Warning>
